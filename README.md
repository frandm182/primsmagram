# prismagram
Instagram clone with Express + Prisma + React + React Native


## User Stories

- [] Like / Unlike photo
- [] Comment on a photo
- [] Search by user
- [] Search by location
- [] See user profile
- [] Follow / Unfollow User
- [] See the full photo
- [] Edit my profile
- [] Create account
- [] Log in
- [] Upload photo
- [] Edit the photo
- [] See the feed
